# ROS 2 Package Development Lab

## Learning Objectives
- Create a new ROS 2 package
- Implement basic publisher and subscriber nodes
- Test the package functionality

## Lab Instructions
This lab provides hands-on experience with ROS 2 package development and basic communication patterns.

## Prerequisites
- ROS 2 installed and sourced
- Basic understanding of ROS 2 concepts

## Expected Outcomes
- A functional ROS 2 package with publisher and subscriber nodes
- Understanding of ROS 2 build system

## Assessment Criteria
- Package builds without errors
- Publisher and subscriber nodes communicate correctly
- Code follows ROS 2 best practices
